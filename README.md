# AI-Legal-Assistant# ⚖️ AI Legal Assistant

An AI-powered Legal Assistant that analyzes crime scenarios under Indian law and provides relevant legal sections, punishment details, and simplified explanations in multiple Indian languages. The application also supports speech-to-text and text-to-speech for a hands-free experience.

## 🚀 Features

- 📝 Analyze crime stories using AI
- ⚖️ Suggest relevant Indian legal sections
- 📚 Display punishment summary
- 💡 Simplified explanation for easy understanding
- 🌐 Multilingual support
  - English
  - Kannada
  - Hindi
  - Tamil
  - Telugu
  - Malayalam
  - Marathi
  - Gujarati
  - Bengali
  - Punjabi
- 🎤 Speech-to-Text
- 🔊 Text-to-Speech
- 🎨 Modern responsive UI with glassmorphism

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/AI-Legal-Assistant.git

cd AI-Legal-Assistant
```

### Create Virtual Environment

Windows

```bash
python -m venv venv
venv\Scripts\activate
```

Linux/macOS

```bash
python3 -m venv venv
source venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

## 📌 API Endpoints

### Analyze Crime

```
POST /analyze
```

Request

```json
{
  "story":"A person steals a motorcycle.",
  "output_language":"English"
}
```

---

### Speech to Text

```
POST /speech-to-text
```

Upload WAV audio file.

---

### Text to Speech

```
POST /text-to-speech
```

Request

```json
{
"text":"Legal analysis...",
"language":"en"
}
```
